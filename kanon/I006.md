# I006 — Klasyfikator warstwowy i macierz stanów obsługi

## Funkcja
Krótki słownik używany przez P008. Nie interpretuje treści, nie podejmuje decyzji i nie zastępuje audytu. Ujednolica cztery niezależne wymiary: warstwa, stan obsługi, sterowanie i flaga.

Stan obsługi materiału nie jest statusem dokumentu w repozytorium. Nie zastępuje `status_produkcyjny` ani `status_epistemiczny` z S002.

## Warstwy W0–W9
- W0 — wejście surowe
- W1 — bramka / routing
- W2 — core / pień
- W3 — alarm / znaczenie / ryzyko
- W4 — nawyk / skrypt / automatyzm
- W5 — ciało / budżet biologiczny
- W6 — pamięć głęboka / archiwum / geneza
- W7 — pole robocze / analiza / redakcja
- W8 — strategia / audyt / hierarchizacja
- W9 — odpowiedź / decyzja / działanie

## Stany obsługi A–E
- A — aktywny i gotowy do użycia
- B — wartościowy, wymagający integracji lub walidacji
- C — nierozstrzygnięty, nie steruje systemem
- D — używany wyłącznie jako źródło, przykład, geneza lub stara wersja
- E — wyłączony z bieżącej obsługi

Stan obsługi nie opisuje ważności ani prawdziwości materiału. Opisuje wyłącznie sposób traktowania go w procedurze P008.

## Sterowanie S0–S4
- S0 — nie steruje
- S1 — działa tylko jako przykład lub kontekst
- S2 — działa dopiero po walidacji
- S3 — działa jako ostrzeżenie lub warunek brzegowy
- S4 — może bezpośrednio wpływać na odpowiedź albo decyzję

Materiał może być ważny i jednocześnie mieć S0 lub S1.

## Flagi
- AMBIG — wejście niejasne; doprecyzuj zamiast zgadywać
- IMG — wymagany plik lub obraz; sprawdź dostęp
- WEB — wymagana aktualna wiedza; użyj świeżych źródeł
- PRIV — dane prywatne; minimalizuj, redaguj, anonimizuj
- LEGAL — prawo lub formalna procedura; sprawdź aktualność i granice
- MED — zdrowie lub leczenie; zachowaj ostrożność, oddziel logistykę od diagnozy
- P009 — ostre ryzyko; zatrzymaj tryb lekki
- COPY — cudza treść chroniona; ogranicz reprodukcję
- FRAME — materiał zawiera możliwe narzucenie ramy interpretacyjnej; sklasyfikuj podtyp i zneutralizuj przed użyciem

## Podtypy FRAME
- ZAŁOŻENIE — nieudowodniona teza ukryta w pytaniu albo poleceniu
- ETYKIETA — przypisanie osobie cechy, roli albo tożsamości
- MOTYW — przypisanie ukrytej intencji lub celu
- WINA — przyjęcie odpowiedzialności przed analizą przyczynową
- PRESJA — wymuszenie wyznania, zmiany, tempa albo określonej odpowiedzi
- FAŁSZYWA_ALTERNATYWA — zawężenie możliwych wyjaśnień do narzuconego zestawu
- MORALIZACJA — zamiana problemu operacyjnego w ocenę charakteru
- ZAWŁASZCZENIE — odebranie osobie prawa do własnego opisu doświadczenia

Podtypy FRAME mogą występować łącznie. Samo wykrycie podtypu nie rozstrzyga jeszcze, że rama jest fałszywa; uruchamia obowiązek oddzielenia danych od założenia.

Priorytet bezpieczeństwa ma flaga P009, niezależnie od pozostałych.

## Etykieta
Format: `[WARSTWA / STAN OBSŁUGI / STEROWANIE / FLAGI]`.

Przykład: `[W1 / C / S0 / FRAME:ZAŁOŻENIE+WINA]`.

## Relacja z P008
I006 definiuje wyłącznie etykiety. P008 określa reguły ich przydzielania, prowadzi procedurę i podejmuje decyzję. Jedynym źródłem reguł routingu jest P008; I006 nie rozwija równoległego protokołu.
